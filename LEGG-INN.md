# Regninga uten søkemotor

Denne oppdateringen fjerner Banken og søkefeltet fra forsiden.

## Slik installerer du

1. Kopier `index.html`, `sitemap.xml` og mappen `assets` til roten av GitHub-prosjektet.
2. Godta overskriving av eksisterende filer.
3. Slett disse gamle filene og mappene manuelt:
   - `banken/`
   - `assets/banken.js`
   - `assets/banken-config.js`
   - `assets/home-search.js`
4. Commit og push endringene til GitHub.

Forsiden vil igjen vise kalkulatorene direkte, uten søkefelt eller Banken-lenker.
